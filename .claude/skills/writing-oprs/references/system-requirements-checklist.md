# System Requirements Checklist for OPRs

This checklist provides a quick reference for what to include when documenting requirements for each major building system in an Owner Project Requirements (OPR) document.

Use this as a starting point - not every item applies to every project. Customize based on project type, owner priorities, and system complexity.

---

## HVAC Systems

### General
- [ ] Spaces served and square footage
- [ ] Operating schedule (occupied/unoccupied hours)
- [ ] Design conditions (outdoor and indoor)
- [ ] Expected occupancy and loads

### Temperature Control
- [ ] Heating setpoints by space type (with tolerance)
- [ ] Cooling setpoints by space type (with tolerance)
- [ ] Setback/setup temperatures during unoccupied periods
- [ ] Warm-up/cool-down requirements (optimal start)
- [ ] Local control and adjustment capability

### Humidity Control
- [ ] Relative humidity range (if controlled)
- [ ] Dehumidification requirements
- [ ] Humidification requirements (if applicable)

### Ventilation and Air Quality
- [ ] Outdoor air ventilation rates (reference ASHRAE 62.1)
- [ ] Filtration requirements (MERV rating)
- [ ] CO₂ monitoring and limits (if applicable)
- [ ] Demand-controlled ventilation (if applicable)
- [ ] Exhaust requirements for special spaces

### Pressurization
- [ ] Space pressure relationships (positive/negative)
- [ ] Differential pressure targets between spaces
- [ ] Critical areas requiring pressure control

### Equipment Performance
- [ ] Heating capacity requirements
- [ ] Cooling capacity requirements
- [ ] Equipment efficiency targets (EER, COP, kW/ton, AFUE, etc.)
- [ ] Part-load performance requirements
- [ ] Turndown capability

### Noise and Vibration
- [ ] Maximum noise criteria (NC rating or dBA)
- [ ] Vibration isolation requirements
- [ ] Sound attenuation for ductwork

### Controls and Integration
- [ ] Control system platform and protocols
- [ ] Integration with building automation system
- [ ] Occupancy sensor integration
- [ ] Remote monitoring and control requirements
- [ ] Alarm and notification requirements

### Maintenance and Access
- [ ] Equipment accessibility requirements
- [ ] Filter change access
- [ ] Service clearances
- [ ] Preferred manufacturers or standardization
- [ ] Expected maintenance frequency

### Special Considerations
- [ ] Emergency operation requirements
- [ ] Smoke control or life safety integration
- [ ] Process loads or special cooling requirements
- [ ] Clean room or laboratory requirements (if applicable)

- [ ] Main service metering

- [ ] Data logging and trending
- [ ] Integration with energy management system

### Special Systems

- [ ] Process power requirements

## Commissioning-Specific Items

For each system, define:
- [ ] Acceptance criteria (measurable performance targets)
- [ ] Testing requirements (functional tests needed)
- [ ] Sampling strategy (100% vs. representative sample)
- [ ] Seasonal or deferred testing needs
- [ ] Training requirements for operations staff

---

## Using This Checklist

**During Owner Interviews:**
- Use as a conversation guide to ensure all systems are discussed
- Identify which items are priorities vs. standard requirements
- Note where owner has specific preferences or past experience

**During OPR Development:**
- Check off items as they're documented in the OPR
- Flag items that need more information or owner clarification
- Verify measurable criteria are established for key requirements

**During Design Reviews:**
- Compare Basis of Design against these OPR requirements
- Ensure all owner requirements are addressed
- Identify any gaps or conflicts to resolve

**During Commissioning:**
- Use as basis for developing test procedures
- Verify all critical requirements are tested
- Confirm documentation and training cover all necessary systems
